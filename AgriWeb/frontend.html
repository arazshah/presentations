نکات پیاده‌سازی

React.js / Frontend

  • استفاده از Tailwind CSS برای استایل‌دهی سریع
  • کامپوننت‌های Headless UI برای دسترسی‌پذیری
  • نقشه با Leaflet یا OpenLayers
  • ترسیم با Leaflet.draw یا Turf.js
  • مدیریت state با Zustand یا Redux Toolkit
  • فرم‌ها با React Hook Form + Zod

NestJS / Backend

  • معماری ماژولار برای هر دامنه
  • PostGIS برای داده‌های مکانی
  • TypeORM با پشتیبانی Geometry
  • احراز هویت با JWT + Passport
  • مجوزدهی با CASL یا Guards
  • مستندات با Swagger
