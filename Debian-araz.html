🧠 مفاهیم کلیدی Tile Server

🔍 Tile Server چیست؟

سرویسی که نقشه‌ها را به صورت کاشی‌های کوچک (256x256 پیکسل) تقسیم کرده و بر اساس درخواست کاربران ارائه می‌دهد

🎨 نحوه کار:

  • نقشه به کاشی‌های کوچک تقسیم می‌شود
  • هر کاشی مختصات Z/X/Y دارد
  • Z = سطح زوم (0 تا 18+)
  • X,Y = موقعیت کاشی در شبکه
نمونه URL کاشی:
/tiles/12/2048/1536.png
↑     ↑   ↑    ↑
style zoom x   y

🐳 نصب و پیکربندی Docker

📦 نصب Docker از مخزن رسمی

# حذف نسخه‌های قدیمی
sudo apt remove docker docker-engine docker.io

# نصب وابستگی‌ها
sudo apt install -y \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg \
    lsb-release

# اضافه کردن کلید GPG
curl -fsSL https://download.docker.com/linux/debian/gpg | \
sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

# اضافه کردن مخزن Docker
echo "deb [arch=$(dpkg --print-architecture) \
signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] \
https://download.docker.com/linux/debian \
$(lsb_release -cs) stable" | \
sudo tee /etc/apt/sources.list.d/docker.list

🚀 راه‌اندازی TileServer GL

📁 ایجاد ساختار پروژه

# ایجاد دایرکتوری پروژه
mkdir ~/tileserver-project
cd ~/tileserver-project

# ایجاد ساختار فولدرها
mkdir -p {data,config,styles,fonts}

# دانلود TileServer GL
git clone https://github.com/maptiler/tileserver-gl.git
cd tileserver-gl

🔧 فایل Docker Compose

# docker-compose.yml
version: '3.8'
services:
  tileserver:
    image: maptiler/tileserver-gl:latest
    ports:
      - "8080:8080"
    volumes:
      - ./data:/data
      - ./config:/config
    environment:
      - MBTILES_DIR=/data
    restart: unless-stopped

🎯 راه‌اندازی سرویس

# اجرای سرویس
docker compose up -d

# بررسی وضعیت
docker compose ps
docker compose logs tileserver

# تبدیل GeoJSON به MBTiles

    --maximum-zoom=14 \
    --minimum-zoom=0 \
    --drop-densest-as-needed \
    --extend-zooms-if-still-dropping \
    iran.geojson

⚙️ پیکربندی Reverse Proxy

# /etc/nginx/sites-available/tileserver
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
    
    # کش کردن کاشی‌ها
    location ~* \.(png|jpg|jpeg|gif|ico|svg)$ {
        proxy_pass http://localhost:8080;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}

⚡ بهینه‌سازی و مانیتورینگ

🚀 بهینه‌سازی عملکرد

# تنظیمات کش در Nginx
location /tiles/ {
    proxy_pass http://localhost:8080;
    proxy_cache tiles_cache;
    proxy_cache_valid 200 30d;
    proxy_cache_valid 404 1m;
    add_header X-Cache-Status $upstream_cache_status;
}

# تعریف کش
proxy_cache_path /var/cache/nginx/tiles 
    levels=1:2 
    keys_zone=tiles_cache:10m 
    max_size=1g 
    inactive=30d;

🌐 ادغام با OpenLayers

// JavaScript
import Map from 'ol/Map';
import View from 'ol/View';
import TileLayer from 'ol/layer/Tile';
import XYZ from 'ol/source/XYZ';

const map = new Map({
  target: 'map',
  layers: [
    new TileLayer({
      source: new XYZ({
        url: 'http://localhost:8080/styles/basic/{z}/{x}/{y}.png'
      })
    })
  ],
  view: new View({
    center: [5704629.865282, 4260114.963312],
    zoom: 10
  })
});
