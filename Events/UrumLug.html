🧩 سه جزء اصلی

💎
Domain (هسته)
منطق خالص کسب‌وکار - بدون هیچ وابستگی خارجی
  • Entities (موجودیت‌ها)
  • Value Objects
  • Domain Services
  • Business Rules
🔌
Ports (درگاه‌ها)
قراردادها (Interfaces) برای ارتباط با دنیای بیرون
  • Input Ports (Use Cases)
  • Output Ports (Repositories)
  • Abstract Interfaces
  • Protocol Definitions
🔧
Adapters (آداپتورها)
پیاده‌سازی‌های واقعی برای ارتباط با فناوری‌ها
  • REST Controllers
  • Database Repositories
  • Email Services
  • External API Clients
