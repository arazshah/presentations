🔓 نرم‌افزار آزاد چیست؟

چهار آزادی ضروری:

  • آزادی ۰: اجرای برنامه برای هر هدفی
  • آزادی ۱: مطالعه و تغییر کد منبع
  • آزادی ۲: توزیع مجدد کپی‌ها
  • آزادی ۳: توزیع نسخه‌های بهبودیافته

"آزاد مثل آزادی بیان، نه آبجو رایگان!"

نرم‌افزار آزاد به کاربران این امکان را می‌دهد که نه تنها از نرم‌افزار استفاده کنند، بلکه آن را مطالعه، تغییر و بهبود دهند. این فلسفه بر پایه همکاری و اشتراک‌گذاری دانش بنا شده است.

💰 صرفه اقتصادی

نرم‌افزار آزاد نه تنها از لحاظ اقتصادی مقرون‌به‌صرفه است، بلکه امنیت بیشتری نیز ارائه می‌دهد. وقتی کد منبع در دسترس عموم است، هزاران چشم آن را بررسی می‌کنند و باگ‌ها سریع‌تر کشف و رفع می‌شوند.

🐍 PyPI - فروشگاه پایتون

Python Package Index

مخزن مرکزی بسته‌های پایتون با بیش از ۴۰۰,۰۰۰ پروژه!

# نصب یک بسته از PyPI pip install requests # جستجوی بسته‌ها pip search django # لیست بسته‌های نصب شده pip list # به‌روزرسانی بسته pip install --upgrade requests
  • دسترسی آسان به هزاران کتابخانه
  • مدیریت خودکار وابستگی‌ها
  • نصب و به‌روزرسانی ساده
  • پشتیبانی از نسخه‌های مختلف

PyPI قلب اکوسیستم پایتون است. هر روز میلیون‌ها دانلود از این مخزن انجام می‌شود و توسعه‌دهندگان از سراسر جهان بسته‌های خود را در آن منتشر می‌کنند.

📦 انتشار بسته در PyPI

1 ساختاربندی پروژه: ایجاد فایل‌های pyproject.toml، README.md و LICENSE
2 نصب ابزارها: pip install build twine
3 ساخت بسته: python -m build
4 تست در TestPyPI: twine upload --repository testpypi dist/*
5 انتشار نهایی: twine upload dist/*
# ساختار پروژه my_project/ ├── src/ │ └── my_package/ │ ├── __init__.py │ └── main.py ├── tests/ ├── LICENSE ├── README.md └── pyproject.toml

انتشار بسته در PyPI فرآیندی ساده اما قدرتمند است که به شما امکان اشتراک‌گذاری کد خود با جامعه جهانی پایتون را می‌دهد.
